Abra o Prompt de Comando (CMD) no Windows.

Pressione Win + R → digite cmd → Enter.

Vá até a pasta do projeto onde está o seu server.js:

cd C:\Users\Bruno Trader Pro\investor-api


Inicie o servidor com Node.js:

node server.js
